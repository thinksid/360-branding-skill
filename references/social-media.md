# 360xRunning — Social Media Reference

> Loaded by /360-branding when the output is a social media asset.

## Formatos

| Plataforma | Tipo | Dimensiones |
|------------|------|-------------|
| Instagram | Post cuadrado | 1080×1080 px |
| Instagram/TikTok | Story / Reel | 1080×1920 px |
| LinkedIn | Post | 1200×627 px o 1080×1080 px |

## Reglas de diseño

- Tipografía: Poppins únicamente
- Colores: solo paleta de marca
- Logo siempre visible, respetando área de protección
- Tono: directo, inspiracional, de "tú"
