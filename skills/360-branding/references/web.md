# 360xRunning — Web Reference

> Loaded by /360-branding when the output is a web page or landing page (HTML + CSS).

## Stack

- HTML + CSS (zero dependencies, inline o single file)
- Google Fonts: Poppins
- Responsive: mobile-first

## Colores CSS

```css
:root {
  --rosa: #F1ACA5;
  --crema: #F1F1E8;
  --terracota: #A8564D;
  --verde: #163935;
}
```

## Reglas de diseño web

- Tipografía: Poppins únicamente (importar desde Google Fonts)
- Solo colores de la paleta de marca
- Logo en header, respetando área de protección
- Tono: directo, inspiracional, de "tú"
